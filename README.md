# SwiftDiagram

SwiftDiagram is a Swift-native modeling language and toolchain for authored architecture diagrams and Swift source inspection.

This repository is currently at Milestone 0: package foundation, model scaffolding, documentation skeletons, fixtures, and CI. Parser, renderer, CLI, configuration loading, and SwiftSyntax inspection behavior are intentionally not implemented yet.

## Current Scope

- Swift package target structure
- Initial semantic model types
- Renderer-neutral interfaces
- Documentation skeletons
- Golden fixture directories

## Build

```bash
swift build
swift test
```
