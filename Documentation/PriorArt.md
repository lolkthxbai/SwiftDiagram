# Prior Art

Status: reviewed for the Milestone 3 PlantUML renderer.

SwiftPlantUML is relevant prior art for PlantUML conventions, filtering, configuration ideas, and golden-file testing. SwiftDiagram does not fork its architecture: the renderer implemented here consumes SwiftDiagram's renderer-neutral semantic model and was written independently against PlantUML's documented class-diagram syntax. No SwiftPlantUML implementation was copied or adapted.
